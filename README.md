# FastAPI BitNet Inference

A web application for running BitNet model inference using FastAPI for the backend and HTML/CSS/JS for the frontend. This application allows users to interact with 1-bit LLMs (like BitNet b1.58) through a simple web interface.

## Features

- FastAPI backend for model inference
- Responsive HTML/CSS/JS frontend
- Real-time text generation
- Supports BitNet models for efficient inference on CPUs

## Requirements

- Python 3.9+
- FastAPI
- Uvicorn
- BitNet.cpp dependencies

## Setup

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Download a BitNet model
4. Run the application: `python app.py`

## Usage

After starting the application, navigate to http://localhost:8000 in your web browser to access the interface.
